| 기호 | 의미 | 키보드 위치 |
|------|------|-----------|
| **⌘** | Command | 왼쪽/오른쪽 아래 (클로버 모양) |

### 기호 조합 예시

- **⌘A** = Command + A = 모두 선택
- **⇧⌘P** = Shift + Command + P = 명령 팔레트
- **⌘K ⌘S** = Command 누르고 K, 다시 S = (2단계 단축키)

## 💡 실전 팁

### 1. **2단계 단축키 (Chord Shortcuts)**

일부 단축키는 두 번의 키 조합으로 동작합니다:

```
⌘K ⌘S → Keybindings 열기
⌘K ⌘F → 문서 서식
⌘K ⌘0 → 모든 영역 접기
```

**팁**: 첫 번째 조합 후 VS Code가 두 번째 입력을 기다립니다.

### 2. **자동 완성 가속화**

```
⌃Space  = 강제 IntelliSense 활성화
⌘.      = 빠른 수정 (Code Action) 메뉴
```

### 3. **선택 및 편집 조합**

```
⌘D를 여러 번 누르기 → 같은 단어 점진적 선택
⌘K ⌘C → 주석 추가
⌘K ⌘U → 주석 제거
```

### 4. **빠른 네비게이션**

```
⌘P     → 파일 검색 (가장 자주 사용)
⌘R     → 함수/메서드로 이동
⌘G     → 특정 줄로 이동
F12    → 정의로 이동
```

## 🚨 단축키 충돌 해결

### 문제 진단

```
1. ⇧⌘P (명령 팔레트) 열기
2. "Developer: Toggle Keyboard Shortcuts Troubleshooting" 입력
3. Output 패널에서 키 입력 로그 확인
```

### 충돌 확인 및 수정

```
1. ⌘K ⌘S (Keybindings 열기)
2. 오른쪽 위의 "Keybindings (JSON)" 버튼 클릭
3. keybindings.json에서 직접 수정
```

### 일반적인 충돌

| 원인 | 해결 방법 |
|------|---------|
| macOS 시스템 단축키 | 시스템 설정에서 VS Code 단축키 비활성화 |
| 다른 애플리케이션 | 해당 앱의 단축키 변경 |
| 확장 프로그램 | 확장 프로그램 비활성화 후 확인 |

## ⚙️ 키 바인딩 커스터마이징

### keybindings.json 수정

```json
[
  {
    "key": "cmd+shift+m",
    "command": "editor.action.copyLinesDownAction",
    "when": "editorTextFocus"
  },
  {
    "key": "cmd+shift+n",
    "command": "editor.action.insertCursorAbove",
    "when": "editorTextFocus"
  }
]
```

### 열기 방법

```
⌘K ⌘S → 오른쪽 상단 "Open Keyboard Shortcuts (JSON)" 클릭
또는
⇧⌘P → "Open Keyboard Shortcuts (JSON)" 검색
```

### 자주 추가하는 커스텀 단축키

```json
// Python 터미널 실행
{
  "key": "cmd+shift+r",
  "command": "python.execInTerminal",
  "when": "editorLangId == python && editorTextFocus"
}

// 파일 저장 후 실행
{
  "key": "cmd+enter",
  "command": "runCommands",
  "args": {
    "commands": ["workbench.action.files.save", "python.execInTerminal"]
  }
}
```

## 📝 마지막 팁: "⌘K"는 마법의 키

VS Code에서 `⌘K` 다음에 다양한 키를 조합하면:

| 조합 | 기능 |
|------|------|
| ⌘K ⌘S | 키 바인딩 열기 |
| ⌘K ⌘F | 문서 서식 |
| ⌘K ⌘0 | 모든 영역 접기 |
| ⌘K ⌘J | 모든 영역 펼치기 |
| ⌘K ⌘C | 커밋 |
| ⌘K ⌘W | 탭/폴더 닫기 |

🎯 **⌘K 다음에 뭘 누를지 모르면**: ⇧⌘P로 명령 검색!
